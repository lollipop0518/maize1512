## 项目简介

本项目是基于玉米育种领域产出数据的分析及预测平台。只在通过流程化的脚本文件及示例数据让非从事数据分析行业的育种人士快速上手并实践数据分析。

#### 功能简介

> 多模块数据分析

- **GWAS**
- **GS**
- **Phenotype data processing**
